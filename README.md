# Звіт з навчальної практики з WEB-програмування

## Тема роботи
Розробка веб-застосунку для обліку особистих фінансів (Budget Tracker)


## Мета роботи
Навчитись створювати веб-застосунок з використанням HTML, CSS та JavaScript, реалізувати збереження даних та взаємодію з користувачем.


## Опис проєкту

У цій роботі я розробив веб-застосунок для обліку особистих фінансів.  
Користувач може додавати доходи та витрати, переглядати баланс, а також аналізувати свої витрати за допомогою графіка.

Дані зберігаються у localStorage, тому після оновлення сторінки інформація не зникає.


## Функціональні можливості

- Додавання доходів та витрат  
- Вибір категорії операції  
- Можливість створити власну категорію  
- Автоматичний підрахунок балансу  
- Відображення загального доходу та витрат  
- Видалення операцій  
- Збереження даних у localStorage  
- Відображення статистики витрат (Chart.js)


## Використані технології

- HTML — створення структури сторінки  
- CSS — стилізація та адаптивний дизайн  
- JavaScript — логіка роботи застосунку  
- Chart.js — побудова графіка  
- localStorage — збереження даних  


## Структура проєкту

Проєкт складається з трьох основних файлів:

- `index.html` — структура сторінки  
- `style.css` — стилі та дизайн  
- `script.js` — логіка роботи  


## Опис роботи програми

Користувач вводить назву операції, суму, обирає тип (дохід або витрата) та категорію.  
Після натискання кнопки "Додати" операція зберігається та відображається у списку.

Баланс автоматично оновлюється залежно від введених даних.  
Також формується графік витрат за категоріями.

Користувач може видаляти операції, після чого всі дані перераховуються.


## Особливості реалізації

- Реалізовано перевірку введених даних (не можна ввести пусті значення або від’ємну суму)  
- Додано можливість вводити власну категорію  
- Реалізовано анімації для кращого UX  
- Інтерфейс зроблений у сучасному стилі  


## Висновок

У ході виконання роботи я навчився створювати веб-застосунок з використанням JavaScript.  
Також я отримав практичні навички роботи з localStorage, обробкою подій та побудовою інтерфейсу користувача.
